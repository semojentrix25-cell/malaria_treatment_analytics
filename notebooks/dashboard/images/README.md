# Malaria Treatment Analytics Dashboard

## Project Overview

An end-to-end healthcare analytics project built using Python, PostgreSQL and Power BI.

## Tools

- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Workflow

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- KPI Development
- Dashboard Development
- Data Storytelling

## Dataset

- 730 treatment records
- 195 participants
