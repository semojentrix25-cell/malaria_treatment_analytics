# Jupyter Notebooks

This folder contains the analytical workflow for the Malaria Treatment Analytics project.

## Notebook Structure

-- 01_data_cleaning.ipynb – Data cleaning, validation and preprocessing.
- 00_data_profiling.ipynb – Initial dataset profiling and quality assessment.
- 02_exploratory_data_analysis.ipynb – Exploratory analysis and visualization.
- 03_feature_engineering.ipynb – Creation of participant-level analytical features.
- 04_kpi_development.ipynb – KPI calculation and business metric development.
