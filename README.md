# Malaria Treatment Analytics Dashboard

## Project Overview

This project analyzes malaria treatment data to evaluate treatment adherence, medication exposure, data quality, and treatment activity trends.

## Tools Used

- Python
- Pandas
- PostgreSQL
- SQL
- Power BI

## Workflow
1. Data Cleaning
2. Data Profiling
3. Exploratory Data Analysis
4. Feature Engineering
5. KPI Development
6. Dashboard Development
7. Data Storytelling

## Dataset

- 730 treatment records
- 195 participants
